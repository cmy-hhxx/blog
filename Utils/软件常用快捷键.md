# 常用软件的快捷键

## tmux(根据修改过后的传家.tmux.conf)
```text
1. ctrl + a + %, 左右平分两个pane
2. ctrl + a + "", 上下平分两个pane
3. tmux a, 打开上次挂起的会话
4. ctrl + d, 关闭当前pane
5. ctrl + a + d, 挂起当前会话
6. ctrl + a + z, 当前所在的pane全屏
7. ctrl + a + s, 选择其他的session
8. ctrl + a + 方向键, 可以选择其他pane
9. tmux 里复制一段文字的步骤
    a. ctrl + a + [
    b. 鼠标选中文字复制
    c. ctrl + a + ]粘贴
10. 按住shift，选中，ctrl + insert，复制
```
## vim(根据修改过后的传家.vimrc)
```text
1. h键，向左
2. j键，向下
3. k键，向上
4. l键，向右
5. gg，到文档首行
6. G，到文档末行
7. n<space>，光标向右移动n个单位
8. n<enter>，光标向下移动n行
9. /word，查找word这个词
10. 命令模式下，:noh，取消刚刚查找词的高亮
11. v，选中文本
12. d，删除选中的文本
13. y，复制选中的文本
14. yy，复制当前行
15. dd，删除当前行
16. p，如果是y复制的，将复制内容粘贴到光标的下一个位置；如果是yy复制的，粘贴到下一行
17. 查找模式下，n向下查，N向上查
18. 命令模式下，u撤销，ctrl + u 取消撤销
19. 命令模式下，:set paste，设置粘贴模式，取消代码自动缩进
20. 命令模式下，:set nopaste， 设置非粘贴模式，开启代码自动缩进
21. 命令模式下，:set nu，开启行号显示
22. 命令模式下，:set nonu， 关闭行号显示
23. gg=G，格式化所有代码
24. ggdG，删除文档所有内容
25. 0，移到本行行首
26. $，移到本行行末
```
