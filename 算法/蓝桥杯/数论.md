#  数论

## 欧几里得算法




$$
求两个正整数的最大公约数 \\
a\:\% \: b=(b, \:a \% \:b)
$$

```c++
int gcd(int a, int b)
{
    if (b == 0) return a;
    else return gcd(b, a % b);
}
```

## 扩展欧几里得算法

> 裴蜀定理：for any positive integer a, b; (a, b) = d; we can find (x, y) as the coefficients of the equation: a * x  + b * y = d.

```c++
int exgcd(int a, int b, int &x, int y)
{
    if (!b)
    {
        x = 1, y = 0;
        return a;
    }
    int d = exgcd(b, a % b, y, x);
    y -= (a / b) * x;
    return d;
}
```
